PLEASE README




  • Bước 1: Ta truy cập vào đường dẫn: ../getStarted/started.html, và mở file started.html để bắt đầu.
    • (Vì file html này được chèn thêm nhạc trong nền để làm thú vị hơn nên hãy mở nó bằng trình duyệt Firefox, vì những trình duyệt khác không hỗ trợ âm thanh nhúng autoplay)

    Bước 2: Khi đã mở file started.html, có 3 mục chính:
  • Truy cập đến All in One: Đây là mục quan trọng nhất, vì click vào sẽ dẫn đến file chính của project
  • Giới thiệu về nhóm: Hiển thị thông tin các thành viên trong nhóm, có thể xem khái quát được mức độ đóng góp của từng thành viên
  • README: chính là file README này


    • Bước 3: Click vào mục "All in One - Tất cả trong một"
  • Trong file chính này, sẽ biểu diễn 3 giải thuật duyệt đường đi của đồ thị: BFS, DFS và tìm thành phần liên thông (Find Connected Components)
  • Ở bước 4 sẽ biểu diễn BFS và DFS
  • Ở bước 5 sẽ biểu diễn Find Connected Componenst


    • BFS và DFS được biểu diễn trên 2 cái mê cung



    Bước 4: BFS:
  • Nhấn nút chạy, thuật toán sẽ bắt đầu duyệt dường đi
  • Nếu muốn xem lại, nhấn nút Tải lại của trình duyệt hoặc nút


    • DFS:
  • Nhấn nút chạy, thuật toán sẽ bắt đầu duyệt dường đi
  • Nếu muốn xem lại, nhấn nút Tải lại (bên cạnh nút chạy)


    • Bước 5: Tìm thành phần liên thông:
  • Nhấp chuột vào 1 node bất kì để hiển thị tất cả những đường liên thông tới node đó
  • Tải lại trình duyệt để đổi 1 cây đồ thị khác

    • Xem file README.md dự phòng tại đây
